# mealie

Mealie recipe manager stack.

## Stack
- Services: `mealie`, `mealie_db`
- Main image: `hkotel/mealie:latest`
- URL: `http://127.0.0.1:9925`

## Links
- Website: https://mealie.io/
- Docker Hub: https://hub.docker.com/r/hkotel/mealie
- GitHub: https://github.com/mealie-recipes/mealie

## Run
```bash
cp .env.example .env
docker compose up -d
```

## Notes
- `hkotel/mealie` is the Docker Hub image used for this stack.
